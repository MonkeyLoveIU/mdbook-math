1.2 引例1：曲边梯形的面积

问题描述

设 f 为闭区间 [a,b] 上的连续函数，且 f(x)≥0。由以下四条边界围成的平面图形称为曲边梯形：

  • 上边界：曲线 y=f(x)
  • 下边界：x 轴（y=0）
  • 左边界：直线 x=a
  • 右边界：直线 x=b
    y
    │     y=f(x)
    │    ╱‾‾‾‾‾╲
    │   ╱        ╲
    │  ╱          ╲___
    │ ╱               ‾╲
    │╱__________________╲___
    └─────────────────────→ x
    a                    b
    
    图9-1：曲边梯形

问题：如何求这个曲边梯形的面积 S？

解决思路："分割-近似-求和-取极限"

第1步：分割（Partition）

在区间 [a,b] 上任取 n−1 个分点： a=x0​<x1​<x2​<⋯<xn−1​<xn​=b

这些点把 [a,b] 分割成 n 个小区间 [xi−1​,xi​]，i=1,2,…,n。

再用直线 x=xi​（i=1,2,…,n−1）把曲边梯形分割成 n 个小曲边梯形。

    y
    │     y=f(x)
    │    ╱‾‾‾‾‾╲
    │   ╱│ │ │ │╲
    │  ╱ │ │ │ │ ╲
    │ ╱  │ │ │ │  ╲
    │╱___│_│_│_│___╲
    └─────────────────→ x
    a  x₁ x₂ x₃ ... xₙ₋₁ b
    
    图9-2：分割与近似

第2步：近似（Approximation）

在每个小区间 [xi−1​,xi​] 上任取一点 ξi​。

因为 f 连续，当小区间很小时，f 在该区间上的值变化不大，可以近似认为： f(x)≈f(ξi​),x∈[xi−1​,xi​]

用以 ξi​ 为底、f(ξi​) 为高的小矩形的面积近似替代第 i 个小曲边梯形的面积： ΔSi​≈f(ξi​)⋅Δxi​

其中 Δxi​=xi​−xi−1​ 是小区间的长度。

第3步：求和（Summation）

n 个小矩形面积之和作为曲边梯形面积 S 的近似值： S≈i=1∑n​f(ξi​)Δxi​​(1)

第4步：取极限（Limit）

当分点无限增多，且对 [a,b] 无限细分时（即所有小区间长度都趋于零），如果和式 (1) 趋于某个常数，且这个常数与分点 xi​ 和中间点 ξi​ 的选取无关，则自然地，我们把这个常数定义为曲边梯形的面积 S。

数学表达： S=∥T∥→0lim​i=1∑n​f(ξi​)Δxi​

其中 ∥T∥=max{Δxi​} 表示分割的"细度"。

1.3 引例2：变力做功

问题描述

设质点受力 F 的作用沿 x 轴由点 a 移动到点 b，并设 F 处处平行于 x 轴（图9-3）。

    F(x)
     ↑
     │  →→→→→
    ─┼──────────→ x
     a          b
     
    图9-3：变力作用
  • 若 F 为常力，则做功 W=F⋅(b−a)（初中物理公式）
  • 若 F 为变力，即 F=F(x) 连续依赖于位置 x∈[a,b]，如何计算功 W？

解决思路：同样的"四步法"

第1步：分割

把 [a,b] 细分为 n 个小区间 [xi−1​,xi​]，Δxi​=xi​−xi−1​，i=1,2,…,n。

第2步：近似

因为 F(x) 连续，在很小的位移区间上可近似看作常量。在每个小区间 [xi−1​,xi​] 上任取一点 ξi​，有： F(x)≈F(ξi​),x∈[xi−1​,xi​]

第3步：求和

质点从 xi−1​ 位移到 xi​ 时，力 F 所做的功近似为： ΔWi​≈F(ξi​)Δxi​

总功为： W≈i=1∑n​F(ξi​)Δxi​​(2)

第4步：取极限

对 [a,b] 作无限细分时，若和式 (2) 趋于某个常数，则把这个常数定义为变力所做的功 W： W=∥T∥→0lim​i=1∑n​F(ξi​)Δxi​

1.4 共同的数学结构

观察：两个完全不同的问题（几何问题和力学问题），最终都归结为同一种形式的和式及其极限：

∥T∥→0lim​i=1∑n​f(ξi​)Δxi​​

思想方法总结：

  1. 分割：将整体问题分解为局部小问题
  2. 近似：用简单量（常数）近似复杂量（变量）
  3. 求和：将局部解累加为整体的近似解
  4. 取极限：通过无限细分过程得到精确解

这就是微积分的基本思想——"以直代曲"、"无限逼近"。

第二部分：定积分的严格定义 | Rigorous Definition

2.1 区间的分割

定义 1（区间的分割，Partition）

设闭区间 [a,b] 上有 n−1 个点，依次为： a=x0​<x1​<x2​<⋯<xn−1​<xn​=b

它们把 [a,b] 分成 n 个小区间： Δi​=[xi−1​,xi​],i=1,2,…,n

这些分点或这些闭子区间构成对 [a,b] 的一个分割，记为： T={x0​,x1​,…,xn​}或T={Δ1​,Δ2​,…,Δn​}

定义（分割的模，Norm of Partition）

小区间 Δi​ 的长度为 Δxi​=xi​−xi−1​，记 ∥T∥=1≤i≤nmax​Δxi​​

称为分割 T 的模（Norm）。

注：

  1. ∥T∥ 可用来刻画被分割的细密程度

    • ∥T∥ 越小，分割越细密
    • ∥T∥→0 意味着所有小区间长度都趋于零

  2. 分割 T 一旦给出，∥T∥ 就随之确定

  3. 但具有同一细度 ∥T∥ 的分割 T 却有无限多个

    • 例如：∥T∥=0.1，可以有等分分割、不等分割等

2.2 积分和（黎曼和）

定义 2（积分和，Riemann Sum）

设 f 是定义在 [a,b] 上的一个函数。对于 [a,b] 的一个分割 T={Δ1​,Δ2​,…,Δn​}，任取点 ξi​∈Δi​，i=1,2,…,n，并作和式： i=1∑n​f(ξi​)Δxi​​

称此和式为函数 f 在 [a,b] 上的一个积分和，也称黎曼和（Riemann Sum）。

关键观察：

  1. 积分和既与分割 T 有关，又与所选取的点集 {ξ1​,ξ2​,…,ξn​} 有关
  2. 对同一个分割 T，不同的点集选择会得到不同的积分和
  3. 因此，给定 ∥T∥，对应的积分和有无限多个

2.3 定积分的定义

定义 3（定积分，Definite Integral / Riemann Integral）

设 f 是定义在 [a,b] 上的一个函数。如果存在一个确定的实数 J，对于任意给定的正数 ε，总存在某一正数 δ，使得对 [a,b] 的任何分割 T，以及在其上任意选取的点集 {ξi​}，只要 ∥T∥<δ

就有 ​i=1∑n​f(ξi​)Δxi​−J​<ε​

则称函数 f 在区间 [a,b] 上可积或黎曼可积；数 J 称为 f 在 [a,b] 上的定积分或黎曼积分，记作 ∫ab​f(x)dx=J​(3)

其中：

  • f 称为被积函数（Integrand）
  • x 称为积分变量（Variable of Integration）
  • [a,b] 称为积分区间（Interval of Integration）
  • a,b 分别称为这个定积分的下限和上限（Lower and Upper Limits）

2.4 定义的深入理解

ε-δ 语言的完整表述

f 在 [a,b] 上可积⟺∀ε>0,∃δ>0,∀T,∀{ξi​}:∥T∥<δ⟹​i=1∑n​f(ξi​)Δxi​−J​<ε

解读：

  • ∀ε>0：任意给定精度要求
  • ∃δ>0：可以找到相应的分割细度
  • ∀T：对任何分割（不论等分还是不等分）
  • ∀{ξi​}：对任意选取的中间点
  • ∥T∥<δ：只要分割足够细
  • 结果：积分和与 J 的差距小于 ε

关键点：

  1. 一致性：不依赖于特殊的分割方式
  2. 普遍性：不依赖于特殊的点选择
  3. 极限性：通过 ∥T∥→0 实现

2.5 极限记号表示

虽然定义用的是 ε-δ 语言，但我们也常用极限符号来表达定积分： J=∥T∥→0lim​i=1∑n​f(ξi​)Δxi​=∫ab​f(x)dx​(4)

注意：这个极限与通常的函数极限有很大区别！

类型特点对应关系
函数极限 limx→x0​​f(x)每个 x 对应唯一的 f(x)一对一
积分和极限 lim∥T∥→0​∑f(ξi​)Δxi​每个 ∥T∥ 对应无穷多个积分和一对多

这使得积分和的极限要比通常的函数极限复杂得多。

3.1 注1：可积性是函数的分析性质

可积性是函数的又一重要分析性质（类似于连续性、可导性）。

定理 9.3（连续函数可积定理）

若函数 f 在 [a,b] 上连续，则 f 在 [a,b] 上可积。

3.3 注3：定积分与积分变量无关

性质（积分变量的无关性）

定积分作为积分和的极限，它的值只与被积函数 f 和积分区间 [a,b] 有关，而与积分变量所用的符号无关，即： ∫ab​f(x)dx=∫ab​f(t)dt=∫ab​f(θ)dθ=⋯​

原因：在定义中，积分变量 x 只是一个"哑变量"（Dummy Variable），它的具体符号不影响积分和的值。

类比：

  • 求和符号：∑i=1n​ai​=∑j=1n​aj​=∑k=1n​ak​
  • 定积分：∫ab​f(x)dx=∫ab​f(t)dt

第四部分：定积分的计算实例 | Computation Examples

4.1 从定义直接计算的一般步骤

虽然从定义直接计算定积分非常繁琐，但理解这个过程对把握定积分概念至关重要。

标准步骤：

第1步：选择分割（通常选等分）
       xᵢ = a + i·(b-a)/n, i=0,1,...,n
       Δxᵢ = (b-a)/n

第2步：选择中间点ξᵢ
       常见选择：ξᵢ=xᵢ（右端点）
                ξᵢ=xᵢ₋₁（左端点）
                ξᵢ=(xᵢ₋₁+xᵢ)/2（中点）

第3步：写出积分和
       Sₙ = Σf(ξᵢ)Δxᵢ

第4步：计算极限
       ∫ₐᵇf(x)dx = lim[n→∞] Sₙ

4.2 例1：计算 ∫01​x2dx（抛物线下面积）

问题：求在区间 [0,1] 上，以抛物线 y=x2 为曲边的曲边三角形的面积（图9-5）。

    y
  1 │    ╱|
    │   ╱ |
    │  ╱  |y=x²
    │ ╱___|
    └─────→ x
    0     1
    
    图9-5：抛物线下的曲边三角形

解：由注3，因 y=x2 在 [0,1] 上连续，故所求面积为 S=∫01​x2dx=∥T∥→0lim​i=1∑n​ξi2​Δxi​

计算过程：

为求得此极限，在定积分存在的前提下，允许选择某种特殊的分割 T 和特殊的点集 {ξi​}。

第1步：等分分割

取等分分割： xi​=0+i⋅n1−0​=ni​,i=0,1,2,…,n

Δxi​=n1​,i=1,2,…,n

此时 ∥T∥=n1​→0 当 n→∞。

第2步：选择右端点

取 ξi​=xi​=ni​，i=1,2,…,n。

第3步：写出积分和

Sn​=i=1∑n​f(ξi​)Δxi​=i=1∑n​(ni​)2⋅n1​

=i=1∑n​n3i2​=n31​i=1∑n​i2

第4步：利用求和公式

回忆平方和公式： i=1∑n​i2=6n(n+1)(2n+1)​

代入得： Sn​=n31​⋅6n(n+1)(2n+1)​=6n2(n+1)(2n+1)​

=6n22n2+3n+1​=6n22n2+3n+1​

第5步：取极限

∫01​x2dx=n→∞lim​Sn​=n→∞lim​6n22n2+3n+1​

=n→∞lim​62+n3​+n21​​=62+0+0​=31​​

答案：抛物线 y=x2 与 x 轴、直线 x=1 围成的曲边三角形面积为 31​。

验证（用不定积分）： ∫01​x2dx=[3x3​]01​=31​−0=31​✓

（这个验证方法将在学习牛顿-莱布尼茨公式后变得清晰）

常见误区

❌ 错误1：混淆定积分与不定积分 ∫01​x2dx=3x3​✗（不定积分的形式） ∫01​x2dx=31​✓（定积分是数值）

❌ 错误2：认为定积分一定是正数

  • 定积分可以为负（当 f(x)<0 时）
  • 定积分可以为零

❌ 错误3：忽略可积性条件

  • 不是所有函数都可积
  • 连续函数一定可积（定理9.3）

❌ 错误4：混淆积分和与定积分

  • 积分和：依赖于具体的分割和点选择（有无穷多个）
  • 定积分：唯一确定的数值（极限值）

🎓 本节总结语 | Concluding Remarks

定积分概念的建立是数学史上的里程碑。它从实际问题出发，通过"分割-近似-求和-极限"的思想方法，将看似困难的连续量问题（曲边图形面积、变力做功）转化为可计算的极限问题。

三个核心认识：

  1. 定积分是和式的极限：本质是黎曼和 ∑f(ξi​)Δxi​ 当 ∥T∥→0 时的极限
  2. 定积分是确定的数值：与分割方式、点的选择无关，只依赖于被积函数和积分区间
  3. 定积分有明确的几何意义：曲边梯形面积的代数和

虽然从定义直接计算定积分非常繁琐，但理解这个过程对把握定积分的本质至关重要。在后续学习中，我们将发展出高效的计算方法（牛顿-莱布尼茨公式），但定义始终是理论的基石。
